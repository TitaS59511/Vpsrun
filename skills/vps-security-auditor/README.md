# VPS Security Auditor

Skill for automated security auditing of Linux VPS.

## Quick Start

```bash
# Run audit
bash audit.sh

# Run with auto-fix
bash audit.sh fix
```

## What It Checks

- SSH hardening (key-only, root login)
- Firewall (UFW status)
- Auto-updates
- User permissions
- Sensitive file exposure
- Open ports
- Running services
